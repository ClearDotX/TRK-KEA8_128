Detailed Description

The raw API is operating on PDU level and it is typically used to gateway PDUs between CAN and LIN.

Usually, a FIFO is used to buffer PDUs in order to handle the different bus speeds.

LIN_SDD_320

  1. lin_configuration conf
  2. lin_tl_descriptor tl_conf
  3. lin_transport_layer_queue tl_queue
  1. #lin_ifc_configuration

The call returns the status of the raw frame transmission function:
LD_QUEUE_EMPTY : The transmit queue is empty. In case previous calls to
ld_put_raw, all frames in the queue have been
trans-mitted.
LD_QUEUE_AVAILABLE: The transmit queue contains entries, but is not full.
LD_QUEUE_FULL : The transmit queue is full and can not accept further
frames.
LD_TRANSMIT_ERROR : LIN protocol errors occurred during the transfer;
ini-tialize and redo the transfer.
