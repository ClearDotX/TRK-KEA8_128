This call changes the NAD if the node properties fullfil the test specified by id, byte, mask and invert.

Parameters
iii[IN] lin interface handle
NAD[IN] current NAD value of the target node
id[IN] property ID of the target node
byte[IN] byte location of property value to be read from the target node
mask[IN] value for masking the read property byte
invert[IN] value for excluding the read property byte
new_NAD[IN] new NAD value to be assigned when the condition is met
